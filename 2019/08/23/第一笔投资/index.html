第一笔投资

发表于 2019-08-23   |   分类于 随笔

年岁渐长，逐渐有了两点体会，一是社会中财富的重要性（成长的压力放大了金钱对于一个人的影响），另一个是作家蔡崇达在《皮囊》中写的：「我不相信成熟能让我们接受任何东西，成熟只是让我们更自欺欺人」，尽管这两者没有丝毫关系。

之前读《富爸爸，穷爸爸》一书，让我开始正视财务，不能让金钱一味地成为暂时欲望的牺牲品，当然也不能像物品一样搁置在银行卡里换取微薄的利息，想着如何让钱去赚钱，进行一笔投资。之前一直是想着买辆车，因为每次假期回济南之后都不方便，尤其是回老家，所以便寻思入手一辆 suv。后来仔细想了想，我是在北京工作，由于没有北京车牌（且极难申请）还有高昂的停车费，所以买了车也一年中开不多少次，并且车子的贬值和更新换代很快，加上车是消耗品，各种保险保养油费都是支出。与其把十几二十万投到这里损耗掉还不如投资到其他地方来让钱生钱呢，至于回济南时可以选择打车或者临时租车，或者先给我爸买辆便宜的车开着。

之前看在房产软件上看到一个济南 R1 地铁边的房子，价格低的让人吃惊，精装才7700一平，虽然国家出台了各种政策来稳定房价，济南同其他许多一二线城市的房价都在下降，但这个价格也确实太低了，而且还是地铁房，之前也了解到这个片区是要统一规划的，感觉是有升值潜力，于是心里打了小九九。最近回济南，顺便去售楼处看了，问价格已经97平已经到了九千几，110平八千多，在均价一万七八的济南来说也是白菜价。看过之后，发现这个楼盘的优劣如下：

劣势

  1. 位置偏僻，地处济南市区和长清交接，距离济南市区和长清大学城都有一定距离。
  2. 周围缺少配套设施，虽然有规划，但是暂时就只是光秃秃的十几撞楼盘，很是荒凉
  3. 一期期房，需要2021年交房，投资再出手的时间比较长。
  4. 周围有墓地，之后应该会迁走

优势

  1. 价格低，精装楼盘这个价格确实很吸引人
  2. 地铁房，距离地铁站400米，并且有刘长山路延长线，地铁 R2 和 R3 通车后，交通会很便利，之后的道路规划都利好
  3. 开发商、物业、小区绿化率都不错
  4. 规划前景好，形成规模之后很有升值空间，可以共享济南西站和长清大学城的资源。

权衡以上利弊之后，决定入手，济南的地铁才刚起步，目前只通了 R1 线，地铁带来的便利性和红利许多济南市民还没有认识到，想想北京就知道了，是不是地铁房在价格上查了很多。并且地铁通了之后，交通的便利性会减小地域带来的楼盘差价，十几分钟路程不会楼盘单价差很多；另外济南近几年发展迅速，尤其是东部高新区和西边的西客站到长清区，并且楼盘所在的平安镇将打造经济开发区，人流多了之后，住房的需求自然增加。个人认为会有升值潜力，遂决定投资。

当天交了定金，入手110平的，找人走团购，总价九十多万，单价才8160。由于前两年刚在市中区买了一套九十多平的学区房，手头吃紧并且再购买属于二套房需要缴纳60%首付，于是用我爸的名义进行购买，首付三成，不到三十万，分期25年，月供差不多四千。预测三年之后，能到1.2w~1.3万一平，或者时间拉长，随着周围基础设置完善和形成规模，房价增加更多也说不定，毕竟离它紧四站的西客站房价现在都一万五六了。

投资是有风险的，这点我肯定是有心理准备，房价涨了皆大欢喜，不涨的话就让我爸妈住，也尽份孝心。三年之后再看吧，今天种下的种子，到底能不能结出饱满的果实。

这是我步入社会之后第一笔大的投资，希望有一个好的结果，既然决定了，不迟疑，不后悔，不怨天尤人，放心大胆干就是了。
