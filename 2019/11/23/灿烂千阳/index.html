灿烂千阳

发表于 2019-11-23   |   分类于 书评

《灿烂千阳》是美籍阿富汗裔作家卡勒德·胡赛尼继《追风筝的人》之后的又一力作，作品这次把视角放在了战争下悲惨的阿富汗妇女身上，用同样细腻生动的笔触描绘了战争下阿富汗人民的苦难，作者除了一如既往对战争的控诉，还有为妇女权利的呐喊。

小说分两个故事展开，两个故事的女主在最后又奇妙的进行交织，走在了一起，共同经历家庭暴力和忍受战争苦难，最后进行反抗并最终实现了自己的人生价值。

玛丽雅姆是一个「哈拉米」（私生女），与母亲相依为命，离群索居生活在简陋的泥屋，每天母亲给她灌输她们的命运没有幸福可言，只有忍耐这样的话。玛丽雅姆最期待的事情是在每个星期四与生父扎里勒见面，对于她而言，父亲是通往美好世界的联系，给了她对外面世界的向往。在15岁生日那年，玛丽雅姆不顾母亲甚至是以死亡相逼的挽留，离开了泥屋到了父亲面前，却经历了之前母亲的预言，经历了父亲家庭的抛弃，最后母亲自杀，她被迫嫁给中年鞋匠拉希德，之后终日生活在动荡年代的家庭暴力阴影下。

小说的另一位女主人翁莱拉，有相对和睦的家庭和青梅竹马的恋人。而战争摧毁了这座城市，一颗炸弹夺去了她的双亲，恋人塔里克一家逃亡他乡。当她受伤醒来，发现被鞋匠拉希德从废墟中救回家。后来刚失去父母的莱拉被拉希德雇人哄骗，以为恋人塔里克在逃亡中病死医院，绝望的莱拉在发现自己怀了爱人的孩子之后同意嫁给已经50多岁的拉希德，之后同样和玛利雅姆一样受到拉希德暴躁而蛮横专制的对待。

至此，两个女人相互依靠，共同生活在丈夫暴力和冷漠中。之后莱拉和玛丽雅姆开始策划逃跑，但是最终因为社会不允许女性在没有男性陪同下单独外出而失败，这一场失败的逃亡使得莱拉和女儿差点被拉希德活活饿死，玛丽雅姆被打的奄奄一息。后来阿富汗被塔利班占领，战争的残酷席卷他们的生活，此时法律不许女人出外工作，拉希德的鞋店也被烧毁，他们陷入了饥荒，此时莱拉居然见到了恋人塔里克，方知被丈夫欺骗，以为恋人已死。后来他们相见的事情被拉希德知道，他火冒三丈要打死莱拉，玛丽雅姆为救莱拉，用铁锹结束了这个残暴的男人，这是她人生第一次自己决定自己的生活轨迹。为了保护莱拉，玛丽雅姆自己主动接受了死刑，完成了英雄般的自我救赎。莱拉带着孩子投奔了恋人塔里克。

故事发生的背景时代其实离我们生活的时代不远，但是那种战争残酷下阿富汗的民众的生活百态，却让我们不敢相信世界还有这样如此可怖的地方，书中有这样的描绘：

驻扎在山上的士兵以向山下的市民开枪来练枪法，以是否打中为胜负标准下赌注，至于靶子是男人、女人还是小孩，则由他们随机挑选。

就是在这个残酷、绝望、苦难的世界，作者却讲述了一个关于希望、救赎和爱的故事。

在阿富汗的父权至上的社会中，婚姻成了女性苦难的无底深渊，女人成了照顾丈夫的奴役，她们不能出去工作，甚至在塔利班时期，女人出门必须有男人陪同才能出门。她们成为了男人的附属品，成为了私有财产，连最基本的人权都没有，作为人的一面被践踏蹂躏。他们只能裹上布卡，默默忍受这份痛苦。

作者没有只是单纯的去描绘这份苦难，在批判父权专制和战争残酷的同时，也表达出了这个国度里女性坚韧的一面，她们默默承受苦难，无论遭受什么样的摧残，她们都坚强的活着，并且心底仍怀有微薄的希望，她们渴望重获自由和幸福，就像最后玛利雅姆，依然完成了自己的使命，实现了自我价值。

最重要的是，玛丽雅姆就在莱拉自己心中，在那儿，她发出一千个太阳般灿烂的光芒。

最终战争会结束，塔利班政权终被推翻，和平会重新降临，阿富汗的女性也会终得解放重获幸福。阿富汗之后的建设离不开她们，她们是驱走人心底寒冷的灿烂千阳，她们用爱的力量战胜恐惧，也将最终收获一个阳光灿烂的未来。
