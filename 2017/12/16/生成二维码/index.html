生成二维码

发表于 2017-12-16   |   分类于 工具   |     |   本文总阅读量 次

本文介绍了二维码的相关知识，并给出了通过 Java 和 jsp 两种方式来生成二维码的实例。

一、简介

  • 一维码：一维条码是由一组粗细不同、黑白（或彩色）相间的条、空及其相应的字符（数字字母）组成的标记，即传统条码。
  • 二维码：用某种特定的几何形状按一定规律在平面（二维方向）上分布的条、空相间的图形来记录数据符号信息。

二维条码有许多不同的码制，就码制的原理而言，通常分为三种类型：

  • 线性堆叠式二维码
  • 矩阵式二维码
  • 邮政码

其中，矩阵式二维码为主要使用类型：

缺点

  • 二维码技术成为手机病毒、钓鱼网站传播的新渠道
  • 信息泄露

目前流行的三大国际标准：

  • PDF417：不支持中文
  • MD：专利未公开，需支付专利费用
  • QR code：专利公开，支持中文

其中，QR code 是日本 Denso 公司研制一种矩阵二维码符号码，全称是 Quick Response Code。

相比其他二维码，具有识别读取速度快，数据密度大，占用空间小的优势
